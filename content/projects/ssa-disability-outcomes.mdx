# SSA Disability Outcomes Analysis

This project examines disability claim outcomes with a focus on statewide approval rates.

## Focus Areas

- Approval likelihood by state (2021)
- Interactive choropleth map + ranking charts
- Clear methodology and assumptions
- Tableau-ready dataset for map/filters

## Data & Prep Notes

- **Rows/Columns:** 1,092 rows × 30 fields
- **Years covered:** FY2001–FY2021
- **Key rates:** adult/child filing rates, allowance rates, and favorable determination rates
- **Output:** `ssa_disability_tableau_ready.csv` for Tableau/BI workflows

## Workflow Overview

1. **Load raw SSA tables** and validate schema/headers.
2. **Clean numeric fields** (remove commas, cast to numeric).
3. **Exploratory analysis** (trends, state comparisons, COVID marker).
4. **Tableau export** with `State-FY` keys for joins.

## Notebook Highlights

- Approval‑rate trends with a **COVID onset marker (2020)**.
- State ranking tables to highlight top/bottom approvals.
- Choropleth map for FY2021 as the main reporting snapshot.

## Tableau Workbook Details

- Added `State-FY` keys for stable joins across sheets.
- Standardized numeric columns for ranking tables.
- Default view set to FY2021 for choropleth and ranking tables.

## Notebook Snippets

```python
# Clean numeric fields stored as strings with commas
numeric_cols = [
    "SSA Disability Beneficiaries  age 18-64*",
    "Favorable Adult Determinations",
    "All Adult Determinations",
]
for col in numeric_cols:
    df[col] = pd.to_numeric(df[col].astype(str).str.replace(",", ""), errors="coerce")
```

```python
# Tableau-ready export
df["State-FY"] = df["State Code"].astype(str) + "-" + df["Fiscal Year"].astype(str)
df.to_csv("ssa_disability_tableau_ready.csv", index=False)
```

```python
# Normalize headers and clean key dimensions
df["Update Date"] = pd.to_datetime(df["Update Date"], errors="coerce")
df["Fiscal Year"] = pd.to_numeric(df["Fiscal Year"], errors="coerce").astype("Int64")
for col in ["File Name", "Region Code", "State Code", "Date Type"]:
    df[col] = df[col].astype(str).str.strip()
df["State Code"] = df["State Code"].str.upper()
df.columns = (
    df.columns.str.replace("*", "", regex=False).str.replace(r"\\s+", " ", regex=True).str.strip()
)
```
