# How Basic Is Your Name?

This project builds a visual story around SSA baby name data across generations, designed for quick scanning and shareable insights.

## Highlights

- **Generational cohorts** with top-10 names per cohort
- **Trend shifts** from Boomers to Gen Alpha
- **Tableau visuals** optimized for storytelling

## Data Snapshot

- **Rows:** 2,149,477 baby-name records
- **Coverage:** 1880–2024 (145 years)
- **Nulls:** 0 missing values across core fields
- **Top names overall:** James, John, Robert, Michael, William, Mary

## Notebook Snippets

```python
# Concatenate yearly SSA files into a single dataset
dfs = [pd.read_csv(path, names=["name", "sex", "count"], header=None) for path in files]
for year, df in zip(years, dfs):
    df["year"] = year
babynames = pd.concat(dfs, ignore_index=True)
```

```python
# Generation buckets (Tableau-compatible)
babynames["generation"] = np.select(
    [
        babynames["year"].between(1883, 1900),
        babynames["year"].between(1901, 1927),
        babynames["year"].between(1928, 1945),
        babynames["year"].between(1946, 1964),
        babynames["year"].between(1965, 1980),
        babynames["year"].between(1981, 1996),
        babynames["year"].between(1997, 2012),
        babynames["year"].between(2013, 2024),
    ],
    ["Lost", "Greatest", "Silent", "Boomers", "Gen X", "Millennials", "Gen Z", "Gen Alpha"],
    default="Other",
)
```

```python
# Generation ranges used for cohort comparisons
```

## Tableau Workbook Details

- **Calculated fields:** `Decade = INT(FLOOR([year] / 10) * 10)` and `Generation` buckets
- **Generations:** Lost, Greatest, Silent, Boomers, Gen X, Millennials, Gen Z, Gen Alpha
- **Sheets:** Top 10 Boy/Girl names by generation with linked highlights
- **Color mapping:** `F` to #e14f7a and `M` to #a0cbe8 for consistent gender encoding

### Sheet Inventory

- Top 10 Boy Names — Boomers, Gen X, Millennials, Gen Z, Gen Alpha
- Top 10 Girl Names — Boomers, Gen X, Millennials, Gen Z, Gen Alpha

## Notebook Highlights

- **Total births trend:** ~3.7M in 2015, tapering to ~3.33M by 2024
- **Gender distribution:** 1,263,426 female rows vs 886,051 male rows
- **Unisex list:** Jessie, Riley, Casey, Jackie, Johnnie, Peyton, Dakota, Jaime lead balanced usage
